# LLM 提示工程教程 - 中文版

## 项目概述

本项目是 Anthropic Claude 提示工程教程的完整中文翻译版本。所有英文内容已翻译为中文，并添加了详细的中文代码注释。

## 主要内容

### 第五章：格式化输出和为Claude预设响应
- 使用XML标签格式化输出
- 预填充Claude的响应
- JSON格式输出
- 多变量提示模板

### 第六章：预知（逐步思考）
- 逐步思考的重要性
- 思考过程可视化
- XML标签组织思考
- 电子邮件分类实战

## 技术特性

1. **完整中文翻译**：所有markdown文本和代码注释都已翻译为中文
2. **详细代码注释**：每个函数和重要代码段都有详细的中文说明
3. **实践练习**：包含多个实际应用场景的练习
4. **最佳实践**：展示提示工程的核心技术和方法
